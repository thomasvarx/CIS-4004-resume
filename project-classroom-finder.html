UCF Classroom Finder

Description

A group-built application to help students find available study rooms/classrooms at UCF. It shows room availability using a color-coded system and includes a map feature to help users navigate.

Skills Learned

  • Team collaboration and project planning
  • UI/UX thinking for availability visualization (color coding)
  • Basic mapping/navigation feature integration
  • Web/app development workflow (depending on your implementation)
