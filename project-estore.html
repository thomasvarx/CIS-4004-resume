Java E-Store Simulation

Description

A standalone GUI-based e-store application built in Java. Users can add items to a shopping cart, calculate totals with tax, and generate an invoice. The program records transactions by appending them to a persistent log file.

Skills Learned

  • Java programming and object-oriented design
  • GUI development (Java UI concepts)
  • File I/O for persistent logging
  • Basic data handling (cart items, totals, tax)
